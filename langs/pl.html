About Security Warnings

For Windows Users

Since Nyan-FA is not yet registered on the Microsoft Store, Windows will display a security warning only on the first launch. However, this is absolutely not dangerous at all in any way, so if a warning appears, please click "More info" and select "Run anyway".

For Mac Users (How to Fight Apple's Greed)

As a free software developer, I don't pay Apple the $99 annual developer fee. Apple tries to throw software from non-contributing developers into the trash (literally, not metaphorically), so on Mac you need to grant security permission to launch this app.

If a security warning appears and you can't launch the app, please follow these simple steps.

First, after installation, try launching the app. If a security warning appears and you can't launch it, follow these steps: (This operation is only required the first time. After this, not even Apple will be able to bother you anymore.)

1. Open System Settings and click "Privacy & Security"
2. Click "Open Anyway"
